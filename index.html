Research Interests

I am interested in language technologies. The core algorithms and models used in things like machine translation, question answering, opinion analysis and, more broadly, information retrieval.

While at Google, I have managed research and production teams in New York and London that were responsible for a number of innovations used in Translate, Assistant, Cloud and Search.

Specifically, I research:
  • Parsing techniques and complexity for various linguistic formalisms
    • Dependency structures primarily (see MSTParser)
    • Internationalization of resources (see Multilingual Treebank and Universal POS projects)
    • Book: Dependency Parsing
  • Information extraction
    • Opinion extraction and summarization
    • Scientific document parsing, ranking and extraction
  • Machine learning for structured and large-scale natural language processing
  • Algorithms for natural language processing
  • Domain adaptation and cross-lingual projection of linguistic structure
  • Relevant organizations: ACL, SIGDAT, SIGNLL, SIGPARSE, NIPS
  • Relevant journals/publications: Computational Linguistics, Transactions of the ACL, ACL Anthology

Software and Data
Note: I try to answer emails about my software, in particular for MSTParser. However, time constraints limit the amount of requests I can manage.
  • Universal Dependency Treebanks
    Multilingual treebanks annotated/converted in a harmonized scheme.

  • MSTParser
    Implementation of parsers described in ACL and HLT-EMNLP '05 papers.
    • See the new SourceForge page

  • StructLearn
    General online structured learning package.

  • BioTagger
    Conditional Random Field Biomedical Entity Tagger (Genes, Variations and Malignancies). This tagger (or variants of it) form part of the core technology of the following resources:
    • FABLE biomedical literature search engine
    • PlasmoDB relevant literature search

  • MALLET
    In the past I have been known to contribute to MALLET,
    which is a general implementation of Conditional Random Fields
    and other learning algorithms tailored specifically to language.
    Andrew McCallum is the primary writer and caretaker of the package
    which is available at http://mallet.cs.umass.edu
